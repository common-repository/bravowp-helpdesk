A Support Tickets manager, with front-end content, allows vistors to open Support Tickets from your website. WooCommerce integration.

== Description ==
A freeware solid Support Tickets manager for your WordPress portal. With a complete front-end content included, you can allow your
vistors to open Support Tickets directly from your website. Completely build on Twitter Bootstrap it offers a clean and easy-to-
use Administration interface to fully manage all the Helpdesk elements.


= NEW: Email To Tickets =
Fetch Email from your Mailbox (example: support@mycompany.com) and convert them into Helpdesk Tickets.

= NEW: WooCommerce Integration! =
Agents and Customers can select a Product of your WooCommerce Store when creating a Support Ticket.

== Installation ==

* Login as Administrator of your portal;
* Navigate to Plugins -> Add New;
* Click on Upload;
* Select and Upload the .zip file you downloaded from this website;
* Activate the new plugin.
* Short code for Front End: [bravowp-helpdesk-frontend]
